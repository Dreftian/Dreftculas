# Contribuyendo a Dreftculas

Gracias por tu interes en contribuir a Dreftculas! Como proyecto publico, valoramos enormemente las ideas y aportes de la comunidad.

## Como empezar

1. Explora el codigo: Familiarizate con la logica de script.js y el sistema de renderizado en Canvas.
2. Reporta errores: Si encuentras un bug, abre un Issue detallando como reproducirlo.
3. Propon mejoras: Tienes una idea para optimizar el rendimiento o anadir un nuevo efecto? Cuentanos!

## Reglas de estilo

Para mantener la calidad del proyecto, te pedimos:
- Usar nombres de variables descriptivos en ingles (como el codigo base).
- Mantener la compatibilidad con Vanilla JS (sin dependencias).
- Documentar funciones complejas.

## Proceso de Pull Request

1. Asegurate de que tu rama este actualizada con main.
2. Verifica que no haya errores en la consola al ejecutar el proyecto.
3. Describe detalladamente que cambios introduce tu PR.

Esperamos tus aportes!
